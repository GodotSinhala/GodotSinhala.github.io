## Godot හොද තැනකට දාගන්න

අනිත් software වගේ නෙවෙයි. Godot install කරන්න ඕනත් නෑ. නිකන්ම double click කරන්න විතරයි ඕන.  
ඒ නිසා ඔයාට system එකේ ඕන තැනක Godot තියාගන්න පුලුවන්.

හැබැයි මෙන්න මේ step ටික කරලා, Godot normal program එකක් වගේ පාවිච්චි කරන්න පුලුවන්.

* **Program Files** folder එකට යන්න.
* **Godot** කියලා අලුතෙන් folder එකක් හදන්න.
* Godot setup file එක ඒ folder එක ඇතුලට දාන්න.
* ඒ file එක උඩ right click කරලා **Send to → Desktop (create shortcut)** ඔබන්න.

දැන් ඔයාට desktop එකේ ඉදන්ම Godot open කරන්න පුලුවන්.  
Desktop එකේ තියාගන්න කැමති නැත්නම්, ඒක උඩ right click කරලා **Pin to Start** හරි **Pin to Taskbar** හරි දාගන්න පුලුවන්.  
(මං කැමති **Pin to Start** කරලා desktop එක clean එකේ තියාගන්න.)
