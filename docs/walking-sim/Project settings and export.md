# Project Settings හදලා Export කරමු

## Project Settings

**Project settings** ගැන පොඩ්ඩක් කතා කරමු.  
Project → Project Settings වලට යන්න.  
![export](<img/Project Settings and Export/export (1).png>)

**General** tab එකේ, **Configs** යටතේ, අපිට පුලුවන් project name එක, description එක, icons, වගේ දේවල් වෙනස් කරන්න.  
![export](<img/Project Settings and Export/export (2).png>)

**Display → Window** යටතේ, screen size එක, mode එක වගේ display options වෙනස් කරන්න පුලුවන්.  
![export](<img/Project Settings and Export/export (3).png>)

Mode එක **Fullscreen** වලට වෙනස් කරන්න. ඊළගට window එක close කරන්න.  
![export](<img/Project Settings and Export/export (4).png>)

## Export

ඔයාගෙ game එක export කරන්න වෙලාව හරි.  
ඔයා export කරන්නෙ මුල්ම වතාවට නිසා, **Export Templates** download කරන්නත් වෙයි.  

මුලින්ම අපි templates download කරමු.  

**Editor → Manage Export Templates** වලට යන්න.  
![export](<img/Project Settings and Export/e10.png>)

ඔයාට පෙනේවි මේ වගේ message එකක්:  
 **“Export templates are missing. Download them or install from a file.”**  
![export](<img/Project Settings and Export/e11.png>)  
**Download and Install** click කරන්න. ඒක තමයි ලේසිම ක්‍රමය, මොකද අදාල version එක automatically  download වෙනවා.  

මේ වැඩේට active internet connection එකකුයි data ටිකකුයි ඕන. මොකද මේ template ආසන්නව **1 GB** වගේ වෙනවා.
Downlaod වෙලා installation එක ඉවර වුනාට පස්සෙ, window එක close කරන්න.

:::info මේ කරදරේ එක පාරයි ✋
එක පාරයි template download කරන්න ඕන. හැම project එකටම ආයෙ ආයෙ මේක කරන්න ඕන නෑ. අලුත් godot version එකකට මාරු වුණොත් තමයි ආයෙත් template download කරන්න ඕන.  
:::

දැන් තමයි හරියටම export කරන steps ටික පටන්ගන්නෙ.  

Project → Export වලට යන්න.  
![export](<img/Project Settings and Export/export (5).png>)

මුලින්ම, **Export Preset** එකක් add කරන්න. මම තෝරනවා **Windows Desktop**.  
![export](<img/Project Settings and Export/export (6).png>)

**Embed PCK** කියන box එකට tick එකක් දාලා, **Export Project** click කරන්න.  
![export](<img/Project Settings and Export/export (7).png>)

ඊළගට save කරන්න ඕන තැන දෙන්න ඕන. මම **builds** කියලා folder එකක් හදලා ඒක select කරා.  
**Export with Debug** tick එක අයින් කරන්න, ඊළගට **Save** click කරන්න.  
![export](<img/Project Settings and Export/export (8).png>)

දැන් ඔයාට ඔයාගෙ game එක **.exe file** එකක් විදියට තියෙනවා.  
ඔයාගෙ යාලුවොන්ට දෙන්න try කරලා බලන්න. Tutorial එක නම් මෙතනින් ඉවරයි...  
![export](<img/Project Settings and Export/export (9).png>)

තව කියන්න දේවල් ටිකක් තියෙනවා, ඒ ටික ඊළග පිටුවෙන් කියන්නම්.  

දැනට, **Congratulations & enjoy your game 🎉**
