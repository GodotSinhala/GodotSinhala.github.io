මෙතනින් තමයි අලුතින් project create කරන්නෙ/ කලින් හදපු project manage කරන්නෙ.

## Editor Interface එක

දැන් ඔයාට godot editor interface එක ලස්සනට පේනව ඇති. මං ඉක්මනට මේකෙ ප්‍රධානම කොටස් ටික ගැන කියාගෙන යන්නම්.  
දැන්ම මේවා මතක තියාගන්න ඕන නෑ. ඒක ඉබේම වේවි ඉස්සරහට project කරද්දි.

![Editor Interface](img/Editor%20Overview/editor_interface.png)

මේ තියෙන්නෙ ප්‍රධාන කොටස් ටික:

* **Scene Panel** → scene එකේ තියෙන nodes ටික බලාගන්න පුලුවන්.
* **Inspector Panel** → select කරපු node එකකට අදාළ properties බලාගන්න පුලුවන්.
* **FileSystem Panel** → project files ටික පෙන්නනවා.
* **Viewport (මැද area එක)** → අපි game world එක හදන්නෙ මේකෙ.
* **Top Toolbar** → play, pause සහ එක එක tool options තියෙනවා.
* **Bottom Panel** → output එක බලාගන්න, animation කරන්න, debugging, ඒ වගේ ගොඩක් දේවල් කරන්න පුලුවන්.

ගොඩක් ගොඩක් දේවල් තියෙනවා, හැබැයි දැනට ඒවා අදාළ නෑ. අපි basics ටික බලලා ඉමු මුලින්.  
ඉස්සරහට යද්දි දේවල් ගැන ඔයාටම comfortable වෙයි. දැනට ඒවා අවුලක් කරගන්න එපා.

එහෙනම් හදන ගමන්ම ඉගෙනගමු!
